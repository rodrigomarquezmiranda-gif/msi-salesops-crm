Comparar rotación del mismo producto entre proveedores
Cada proveedor describe el mismo producto MSI con su propio texto — el sistema busca el código de modelo real (ej. "A520M-A", "H610M-S") dentro de cada descripción para emparejarlos, aunque el resto del texto no coincida. Se muestran todos los productos con datos cargados. Los que tienen 2+ proveedores permiten comparar directamente entre ellos.
